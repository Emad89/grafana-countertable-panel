# Counter Table Panel

Counter Table Panel extends Grafana's table experience with a configurable counter column.

## Highlights

- Counter column shows `1..N` for the current visible row order.
- Counter updates when the table is sorted or filtered.
- Counter can be shown or hidden.
- Counter header text, width, and position (left/right) are configurable.

## Requirements

- Grafana `>= 12.3.0`

## Configuration

Panel options include:

- Show header
- Show type icons
- Enable pagination
- Frozen columns (left)
- Cell height / max row height
- Disable keyboard events
- Show counter
- Counter header
- Counter position
- Counter width
